# 🌊 Mellow Code Command Center

Selamat datang di Mellow Code. Ini adalah sistem AI murni tanpa bloatware.

## 🤖 Core Modes (Gunakan Sesuai Kebutuhan)
- `/architect <task>` — **(Task Berat)** 2 LLM bekerja: 1 mikir konsep, 1 nulis kode.
- `/code <task>` — **(Task Cepat)** 1 LLM langsung eksekusi.
- `/ask <question>` — **(Task Riset)** Analisa tanpa ubah file.

## 🛠 Mellow Utilities
- `/repo` — Status repository, changes, history.
- `/check` — Scan kualitas kode (TODO, file besar).
- `/skills` — Daftar utility.
- `/models` — Interactive model picker.

## 💎 Prinsip
1. 🪨 **Caveman Mode:** Otomatis aktif. LLM sangat irit bicara, fokus nulis kode.
2. **KISS:** Solusi sederhana lebih baik.
3. **No Bloat:** Tidak ada agen palsu atau lapisan routing tambahan.
